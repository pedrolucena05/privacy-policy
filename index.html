Política de Privacidade

WhatsApp Chatbot - Viver Produções

Esta Política de Privacidade descreve como a Viver Produções trata os dados pessoais dos usuários atendidos por meio do nosso canal de WhatsApp Business, operado com apoio da WhatsApp Business Cloud API.

Dados que coletamos:

  • Número de telefone do WhatsApp;
  • Nome informado no perfil do WhatsApp, quando disponível;
  • Data e horário das interações;
  • Conteúdo das mensagens enviadas pelo usuário;
  • Verifica se o cliente necessita de um atendimento com um de nossos atendentes (não automatizado)

Como usamos os dados

  • Receber, processar e responder mensagens enviadas pelos usuários;
  • Prestar atendimento automatizado;
  • Encaminhar o atendimento para resposta manual, quando necessário;
  • Manter o contexto recente da conversa para continuidade do atendimento;
  • registrar informações mínimas para fins de operação, segurança e suporte.

Modo de funcionamento do app:

  • O chatbot foi desenvolvido apenas para responder mensagens, não fazemos marketing com este app.

Compartilhamento de dados

  • Não vendemos dados pessoais;
  • Os dados podem ser tratados por fornecedores e prestadores de serviço necessários para a operação do canal de atendimento, incluindo a infraestrutura do WhatsApp Business Platform / Meta, além de hospedagem, banco de dados e serviços técnicos relacionados;
  • Também poderemos compartilhar dados quando houver obrigação legal ou determinação de autoridade competente;

Armazenamento:

  • Armazenamos apenas as informações necessárias para o funcionamento do atendimento.
  • São mantidas apenas as últimas 15 mensagens de cada cliente (contando com as que o sistema envia), para o entendimento do contexto para envio das mensagens manuais quando necessário

Segurança:

  • Adotamos medidas técnicas e administrativas razoáveis para proteger os dados pessoais contra acesso não autorizado, perda, alteração, divulgação ou destruição indevida.

Direitos do titular:

Nos termos da Lei Geral de Proteção de Dados Pessoais (LGPD), o titular poderá solicitar, quando aplicável:

  • Confirmação da existência de tratamento;
  • Acesso aos dados pessoais;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Eliminação, bloqueio ou anonimização de dados desnecessários ou tratados em desconformidade com a lei;
  • Portabilidade, quando cabível;
  • Informações sobre compartilhamento;
  • Oposição ao tratamento, quando aplicável.
